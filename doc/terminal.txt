terminal uses vga.
this terminal uses default vga encoding, utf-8 will be in future.
to print characters just use boot::vga::write_bytes to print byte array or boot::vga::write_byte to print one byte (u8).
but there are some control codes that you can use:
0   - prints 10 byte
1   - go to left up corner, home key
2   - go to right down corner, end key
3   - color fg using next hexadecimal character, pgUp
4   - color bg using next hexadecimal character, pgDn
5   - color bg & fg using next binary printed byte format, delete
6   - tab 2 characters, insert
7   - tab 4 characters
8   - tab 6 characters
9   - tab 8 characters
10  - enter
11  - none
12  - end of line
13  - to start of line
14  - print next character to vga (do not check control codes), f1
15  - set color to hightlight, f2
16  - reset color, f3
17  - right
18  - left
19  - up
20  - down
21  - character
22  - set color to be before change, f4
23  - none, f5
24  - switch terminal type (do not change cursor position when printing), f6
25  - switch terminal type (print key presses) // TODO, f7
26  - switch terminal type (buffering text) // TODO, f8
27  - esc (while reading input)
28  - none, f10
29  - shift (while reading input) // TODO
30  - alt (while reading input) // TODO
31  - ctrl (while reading input) // TODO
127 - go to right and remove right character, backspace
